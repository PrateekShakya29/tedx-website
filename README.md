

## Installation

### Requirements

- Node.js v14 LTS or later (with NPM CLI)
- Yarn CLI
- Git CLI

### Procedure

```bash
$ cd website
$ yarn install
```

### To Run

- Development

```bash
$ yarn develop
```

- Testing

```bash
$ yarn test
```

- Production

```bash
$ yarn build
```
